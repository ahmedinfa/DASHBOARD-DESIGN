Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML, innerText, textContent এই তিনটা পদ্ধতি ব্যাবহৃত হয় কোনো HTML Element এর ভিতরের কনটেন্ট পাবার জন্য এবং পরিবর্তন বা সেট করার জন্য। এদের মধ্য মূল পার্থক্য হলোঃ-

innerHTML: কোনো HTML এলিমেন্ট এর মধ্য কি টেক্সট আছে বা অন্য কোনো HTML এলিমেন্ট যদি থাকে তা রির্টান করে।
innerText: কোনো HTML এলিমেন্টের ভিতরের টেক্সট কে রির্টান করে। কিন্তু যদি ওই এলিমেন্টের স্টাইলে display: none; থাকে তাহলে এটা কাজ করবে নয়া।
textContent: কোনো এলিমেন্টের ভিতরে যে কোনো টেক্সট কনটেন্টকে রির্টান করে যদি রে। display: none; থাকে তার পরেও তাকে রির্টান করতে পারে। এটা কমেন্টকেও রির্টান করে।

Question-3: What is event delegation in the DOM?

event delegation একটা পদ্ধতি বা টেকনিক যেখানে কোনো Parent এলিমেন্টের প্রতিটি Child এলিমেন্টকে Event listener না দিয়ে Parent এলিমেন্টকেই Event listener দেওয়া হয়। এটা কোডের পারফরমেন্স বৃদ্ধি করে। এটা ডাইনামিক এলিমেন্টের ইভেন্ট লিসেনর ম্যানেজের ক্ষেত্রে বেশি উপকারি।

Question-4: What is event bubbling in the DOM?

Event bubbling হলো event propagation এর একটি পদ্ধতি যেখানে কোনো চাইল্ড এলিমেন্ট এ কোন ইভেন্ট ঘটলে তা bubble হয়ে ওই এলিমেন্টের প্যারেন্ট এলিমেন্টে পৌছায় এবং তারপর প্যারেন্টের প্যারেন্ট এলিমেন্টেও যায় এভাবে যেতে যেতে উইন্ডো অবজেওক্ট পর্যন্ত যায়। এই ইভেন্ট বাবলিং কে বন্ধ করতে চাইলে event.stopPropagation() ব্যাবহার করতে হবে।

Question-5: How do you create, add, and remove elements using JavaScript?

জাভাস্ক্রিপ্টে বিভিন্ন ভাবে এলিমেন্ট ক্রিয়েট, এড এবং রিমুভ করা যায়। এর মধ্য অন্যতম হলো document.createElement() এর মাধ্যমে এলিমেন্ট বানিয়ে appendChild() এর মাধ্যমে বানানো এলিমেন্টকে এড করা যায় এবং removeChild() এর মাধ্যমে কোনো এলিমেন্টকে আবার রিমুভ করা যায়।
